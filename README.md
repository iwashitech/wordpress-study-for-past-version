# WordPressの学習（昔のWordPress用）

## Usage

```powershell
# start containers
docker-compose up -d

# remove containers
docker-compose down
```

## ページ
### 管理画面
http://localhost:8000/wp-admin/
### ブログページ
http://localhost:8000/
